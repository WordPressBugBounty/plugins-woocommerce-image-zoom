== Frequently asked questions ==

== Frequently Asked Questions ==

= What does this plugin replace in my WooCommerce store? =
This plugin is an all-in-one gallery engine that replaces standalone plugins for image zoom, gallery sliders, product videos, additional variation images, and full-screen lightboxes.

= Do I need coding skills to configure the layouts? =
No coding knowledge is required. You can manage layouts, zoom styles, video integrations, and global rules entirely through an intuitive admin settings interface.

= Is this gallery plugin fully mobile responsive? =
Yes. The plugin features a mobile-first design that supports touch gestures, swipe navigation, and pinch-to-zoom for an optimized mobile shopping experience.

= How do the Gallery Presets and Rules systems work? =
You can build specific layout presets and use the rules system to assign them globally, to selected products, or to specific product categories.

= Can I change the position of the product gallery thumbnails? =
Yes. You can position the thumbnail strip at the top, bottom, left, or right of the main image, or switch between slider and grid structures.

= What are the premium advanced layout modes? =
Upgrading unlocks high-end layouts like the Hierarchy Grid, Scroll-Linked Anchor Navigation, Sticky Split columns, and interactive Tile layouts with show more/less options.

= What types of image zoom effects are included? =
The free version includes high-detail Inner and Hover zoom options, while the premium version introduces advanced Lens Zoom and dedicated Window Zoom panels.

= What interactive tools are available inside the full-screen Lightbox? =
The lightbox includes navigation arrows, image counters, captions, and autoplay slideshows. The premium version adds image rotation, mirror flipping, 1:1 scaling, social sharing, and downloads.

= Which video streaming platforms are supported for product galleries? =
You can embed videos directly into the gallery timeline using YouTube, Vimeo, self-hosted files, or videos from your WordPress Media Library.

= How does the plugin handle variable products? =
When a customer selects a variation, the main gallery instantly replaces or updates the images to match that specific choice. The Pro version supports full multi-image galleries per variation.

= Does this plugin work with any WordPress theme? =
Yes. It integrates seamlessly with popular modern themes, page builders, and core WooCommerce hooks while utilizing a standalone architecture to prevent theme styling conflicts.

= Will adding multiple gallery features slow down my product pages? =
No. The plugin was completely rewritten from the ground up to minimize frontend assets, reduce HTTP requests, eliminate dependency on heavy third-party libraries, and preload images for fast switching.

= How does the AI-powered Product Gallery Assistant work? =
The AI Gallery Assistant helps you create product gallery designs using simple text instructions or reference images. Describe the gallery style you want, upload an example design, and AI will suggest suitable gallery layouts and settings.

= Can I use my own AI API key with this feature? =
Yes. You can connect your own AI provider API key and use AI features directly inside the plugin. This gives you full control over your AI usage and API settings.

= Can AI create a gallery layout from an example image? =
Yes. Upload a reference image of a gallery design you like, and the AI can analyze the style, layout, and structure to suggest a similar product gallery setup.

= What can I ask AI to create for my product gallery? =
You can request different gallery styles, thumbnail positions, slider layouts, zoom experiences, product showcase ideas, and more based on your store's design requirements.

= Can I customize the AI-generated gallery settings? =
Yes. AI provides a starting point, and you can fine-tune every gallery option afterward to match your brand style and shopping experience.

= What are Gallery Quick Start presets? =
Gallery Quick Start provides 12+ ready-made gallery presets that allow you to instantly create different product gallery layouts with a single click. You can customize any preset from the gallery settings afterward.

= Can I use AI and presets together? =
Yes. You can start with a ready-made preset and use AI suggestions to customize or create a gallery style that matches your product presentation needs.
